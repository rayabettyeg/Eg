# رعاية بيتي (RayaBetty) - الموقع الرسمي

هذا المجلد يحتوي على النسخة النهائية والجاهزة تماماً للرفع (Deployment) لموقع "رعاية بيتي".

## محتويات المشروع:
- **ملفات الموقع (HTML/CSS/JS):** صفحات الموقع الرئيسية، الخدمات، الحالات المرضية، المدونة، وصفحة اتصل بنا.
- **ملفات الأرشفة ومحركات البحث (SEO):** 
  - `sitemap.xml`: خريطة الموقع التي تساعد جوجل في أرشفة جميع صفحات الموقع.
  - `robots.txt`: ملف توجيه عناكب محركات البحث.
  - `google-verification.html` (وغيرها): ملفات إثبات ملكية الموقع لـ Google Search Console.
- **مجلد الصور (images):** جميع الصور المستخدمة في الموقع بنسخ عالية الجودة.

## خطوات الرفع (Deployment):
1. قم برفع جميع محتويات هذا المجلد (وليس المجلد نفسه، بل ما بداخله) إلى الخادم الخاص بك (Host / cPanel / Vercel / Netlify).
2. تأكد من أن ملف `index.html` موجود في المسار الرئيسي (Root).
3. بعد رفع الموقع، قم بزيارة Google Search Console وارفع رابط `sitemap.xml` لتسريع الأرشفة.

## ملاحظات:
- تم فحص جميع الروابط الداخلية وهي تعمل بنسبة 100%.
- تم فحص وتصحيح جميع أخطاء الأكواد والمسافات الزائدة.
- تم ضغط وتحسين الصور (بصيغ PNG و JPG) لضمان سرعة تحميل الموقع.
